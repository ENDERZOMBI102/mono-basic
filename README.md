### Building
To configure this module, first configure it, like this:
```shell
$ ./configure [--prefix=/path]
```
To build it, use:
```shell
$ make
```

To install the compiler and runtime, use:
```shell
$ make install
```

### Package
- Visual Basic Compiler
- Visual Basic Runtime
